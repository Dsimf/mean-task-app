## 🎯 Features (Production Ready)
- **Glassmorphism UI** - Modern Tailwind CSS with backdrop-blur, hover effects, smooth animations
- **Reactive Forms** - Form validation, character counter, real-time feedback
- **Task Management** - Add, toggle, delete, filter (all/active/completed), priorities
- **Dark Mode** - Persistent theme toggle with CSS variables
- **Responsive Design** - Mobile-first, works on all devices
- **Performance** - Angular Signals, trackBy optimization, optimized animations
- **Job-Ready Code** - Linted, typed, standalone components, SSR ready

## 🛠️ Tech Stack
```
Frontend: Angular 21 (Standalone) + Tailwind CSS 3.4 + TypeScript 5.9
State: Signals + Reactive Forms
Styling: Tailwind + Glassmorphism + Custom Animations
Future: Express.js Backend + MongoDB
Deployment: GitHub Pages (SSR compatible)
```

## 🚀 Quick Start
```bash
cd frontend
npm install
npm start
```
**Open:** http://localhost:4200/tasks

## 📦 Build & Deploy
```bash
# Build for production

# Deploy to GitHub Pages

```
Live at: https://username.github.io/mean-task-app

## 🧩 Architecture
```
src/app/
├── app.html (Tailwind layout)
├── app.css (deprecated - Tailwind)
├── pages/tasks/
│   ├── tasks.component.html (Tailwind glass UI)
│   ├── tasks.component.ts (ReactiveForms + Signals)
│   └── tasks.component.css (empty)
└── services/
    ├── task.service.ts (Signal state)
    ├── theme.service.ts (Dark mode)
    └── api.service.ts (Backend ready)
```

## 📱 Responsive Breakpoints
- **Mobile** (`<640px`): Stacked layout, touch-friendly buttons
- **Tablet** (`640px+`): Optimized spacing
- **Desktop** (`1024px+`): Full glassmorphism effects

## 🎨 Design System
- **Colors**: Tailwind + Custom glass/opacity
- **Animations**: Custom keyframes (glow, shine, slideIn)
- **Effects**: Backdrop-blur, hover transforms, focus rings
- **Dark Mode**: `dark:` prefix + CSS vars
