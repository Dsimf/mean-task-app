# TaskFlow - Full-Stack Task Management App

![TaskFlow Banner](https://via.placeholder.com/800x200/6366f1/ffffff?text=TaskFlow+-+MEAN+Stack)

A beautiful, fully functional **Task Management App** built with the **MEAN Stack** (MongoDB, Express, Angular, Node.js) as part of a 3-week intensive learning plan.


## ✨ Features
- ✅ Full CRUD operations (Create, Read, Update, Delete)
- ✅ Real-time MongoDB persistence
- ✅ Responsive Tailwind UI with glassmorphism design
- ✅ Search + filters (All / Active / Completed)
- ✅ Demo authentication with localStorage
- ✅ Loading states, error handling, confirmation dialogs
- ✅ Clean separation of frontend & backend

## 🛠 Tech Stack
- **Frontend**: Angular 21 + TypeScript + Tailwind CSS + Reactive Forms + HttpClient
- **Backend**: Node.js + Express + REST API
- **Database**: MongoDB Atlas (Mongoose ODM)
- **Version Control**: Git + GitHub
- **Deployment**: GitHub Pages (frontend) + Render/Railway (backend - coming Day 20)
